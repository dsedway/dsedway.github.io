Scroll Demo

JS Foundation

First, we need to implement a CSS property that indicates the percentage of the page that has been scrolled We use this code in our JS:

window.addEventListener('scroll', () => { document.body.style.setProperty('--scroll', window.pageYOffset / (document.body.offsetHeight - window.innerHeight)); }, false);

Animation Setup

Here is the setup for the animation

svg { display: inline-block; animation: rotate 1s linear infinite; }

@keyframes rotate { to { transform: rotate(360deg); } }

Pausing the Animation

This code determines the duration of the animation relative to the distance scrolled on the page.

svg { position: fixed; /* make sure it stays put so we can see it! */ animation: rotate 1s linear infinite; animation-play-state: paused; animation-delay: calc(var(--scroll) * -1s); }

Extra Animation Code

This helps to ensure the animation works

animation-iteration-count: 1; animation-fill-mode: both;

This is the basics of the code required for an animation, but certain aspects of the animation and the images can be adjusted
